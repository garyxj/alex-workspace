# 自主任务系统 - 使用指南

## 已完成的配置

✅ 创建了 `GOALS.md` - 存储你的所有目标
✅ 创建了 `TASKS.md` - 任务看板（类似 Kanban）
✅ 创建了 Cron Job - 每天 8:00 自动运行（已禁用，待启用）

## 下一步操作

### 1️⃣ 填写你的目标

编辑 `GOALS.md`，在"## 我的目标"部分添加你的目标：

```markdown
### 职业目标
- 将 YouTube 频道增长到 10 万订阅者
- 在 Q3 推出我的 SaaS 产品
- 建立 AI 教育社区

### 个人目标
- 每月读 2 本书
- 学习西班牙语

### 业务目标
- 将收入扩大到 10k/月
- 与 5 家公司建立合作伙伴关系
- 尽可能自动化我的工作流程
```

### 2️⃣ 测试运行

先测试一下系统是否正常工作：

```
"执行一次自主任务生成：读取 GOALS.md，生成 4-5 个可以自主完成的任务，并执行其中一个。"
```

### 3️⃣ 启用自动运行

测试通过后，启用 Cron Job：

```
"启用 Cron Job: Daily Autonomous Tasks"
```

系统会每天早上 8:00 在 Telegram 报告任务完成情况。

## 如何工作

1. **每天 8:00**：系统自动启动
2. **读取目标**：从 `GOALS.md` 获取你的目标
3. **生成任务**：AI 生成 4-5 个可自主完成的任务
4. **执行任务**：AI 执行这些任务（研究、写作、构建等）
5. **更新看板**：在 `TASKS.md` 记录任务状态
6. **报告结果**：在 Telegram 发送完成摘要

## 任务类型示例

- 竞品研究和分析报告
- 基于热点的视频脚本草稿
- 应用功能开发
- 社交媒体内容
- 潜在业务伙伴研究
- 构建一个 MVP 小工具

## 关键原则

1. **完整输出**：提供完整的报告、脚本或代码，而不只是想法
2. **自主执行**：AI 自己完成，不需要你参与
3. **累积学习**：系统会越来越了解你需要什么
4. **可追踪**：通过 `TASKS.md` 查看所有任务历史
