**用户反馈：** 环境里有两个 agent，我是负责工作的，另一个是默认的。配置了两个 bot，另一个不能回复。

**检查结果：**

**当前配置：**
- 主要 bot (token: 8354560182...)
- Work bot (token: 8789774907...)
- Agent "work" 绑定到 Work bot (accountId: 8573621851)

**分析：**
- 当消息通过 Work bot 发送时，由 work agent 处理并回复
- 但当消息通过主 bot 发送时，没有对应的 agent 绑定，主 bot 无法回复

**可能的原因：**
1. 主 bot 没有绑定的 agent 处理其消息
2. 两个 bot 都接收消息，但只有一个 agent 能处理

**已发送建议给用户：**

**方案 A：为主 bot 添加 bindings**
- 将 main agent 绑定到主 bot
- 两个 agent 都能处理各自 bot 的消息

**方案 B：统一路由到 work agent**
- 只保留 work bindings
- 所有消息都由 work agent 处理

**方案 C：检查 main agent 配置**
- 确保 main agent 也有正确的 channel 和 account 配置
