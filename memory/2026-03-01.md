## 错误分析

### 可能的根本原因

**1. OpenClaw 版本问题**
- 当前版本：2026.2.6-3
- Message 工具可能期望不同的配置格式
- 或者版本中有 bug

**2. 配置路径问题**
- 错误信息提到：`TELEGRAM_BOT_TOKEN` 或 `channels.telegram.botToken`
- 但当前配置中，bot token 在 `channels.telegram.accounts.{accountId}.botToken`
- 配置路径不匹配

**3. 内部组件问题**
- Telegram plugin 可能有 bug
- Message 工具可能在查找配置时有问题
- 两个组件之间的接口可能有 bug

**4. 缓存问题**
- OpenClaw 可能缓存了旧的配置
- Gateway 可能没有完全重新加载配置
