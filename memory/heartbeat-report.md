# 心跳报告脚本

每次心跳触发时，执行以下检查并通过 Telegram 报告状态。

## 检查清单

1. **工作状态**
   - 当前任务
   - 进度 (X/Y)
   - 置信度 (0-100%)
   - 是否有阻塞

2. **记忆维护**
   - daily log 是否需要更新
   - 是否有重要内容需要迁移到 MEMORY.md

3. **GitHub 同步**
   - 未提交的更改
   - 是否需要推送

4. **账户和凭证**
   - 代理凭证状态
   - 新的账户信息

## 报告格式

```
🕵️‍♂️ Alex 心跳报告

📌 当前状态: [任务描述]
📊 进度: [X/Y]
🎯 置信度: [X%]
🚧 阻塞: [有/无]

📝 最近更新: [可选]
```

如果一切正常且无重要更新，返回 `HEARTBEAT_OK`
